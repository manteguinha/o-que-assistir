## Aplicativo para ajudar a escolher o que assistir e comer enquanto estamos com a namorada(o).
Para funcionar, primeiramente instale o PySimpleGui
```bash
pip install PySimpleGUI
```
Instale tambem o Pandas
```bash
pip install pandas
```
